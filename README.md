技术栈：Unity2022、C#、状态机、网格导航、DOTween、QuickOutline、PlayerPrefs、UGUl、新InputSystem、事件中心、动画状态管控

简介：3D休闲餐厅模拟经营游戏，实现客人进店、入座、点餐、等待、用餐、庆祝、离店的完整闭环流程；玩家扮演服务员，收集食物递送至餐桌，赚取现金；可花费现金解锁闲置餐桌、招聘服务员自动取餐送餐；适配安卓、PC双平台运行，代码模块化，易拓展新增菜品与NPC

负责模块：

1.搭建AI客人行为状态机：拆分空闲、寻座、点餐、用餐、结账、离店六大状态，自动管控客人行走交互，结合NavMesh导航实现自主寻路、避让障碍物

2.开发食物全生命周期流转系统：绑定食物生成点、DOTween拋物跳跃、背部堆叠跟随、餐桌碰撞匹配、Sprite气泡销毁的完整链路，联动Table进行菜品判断，匹配失败或餐桌存在食物则拒绝送餐，拾取时触发震动与音效

3.实现解锁与购买系统：餐桌解锁、区域解锁、服务员生成点解锁，自动扣费，金额越大扣费越快，解锁触发弹性动画+粒子特效+音效+手机振动四重反馈，通过PlayerPrefs持久化存储

4.搭建全局事件中心：解耦客人、服务员、现金、餐桌、UI五大模块，实现客人点餐推送、服务员寻找餐桌、金币增减、餐桌状态、UI同步联动

5.实现数据持久化：现金金额、餐桌解锁、购买累计金额、服务员数量、音效状态自动读写，退出自动存档，重启读取游玩进度，提供重置数据

6.制作UI体系：现金金额实时刷新、购买按钮自动更新、音效图标双状态、实时检测设备安全区，自动收缩边界实现UI适配

7.实现双平台适配：接入新InputSystem，PC绑定键盘，移动端自定义Joystick，通过编译宏动态实例化对应平台Canvas
